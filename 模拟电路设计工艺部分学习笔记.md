+ 问题1：蚀刻PCB的溶液具有氧化性还是还原性？

**答：一般PCB板工艺流程：线路板--图形电镀铜---图形电镀铅锡--去膜--蚀刻--剥铅锡**

  **蚀刻的概念：先在PCB加工板子外层需保留的铜箔部分，也就是电路的图形部分上预镀一层** ***铅锡抗蚀层***，**然后用化学方式将其余的** ***铜箔*** **腐蚀掉，称为蚀刻。**

  **所以蚀刻的目的就是要把铜箔腐蚀掉，一般蚀刻溶液采用FeCl3（氯化铁）溶液。其与铜发生的化学反应为：**
  
 **<center>FeCl3 + Cu → CuCl + FeCl2 </center>** *其中CuCl具有还原性， 可以和FeCl3进一步反应生成氯化铜。*
  
  **<center>FeCl3 + CuCl → CuCl2 + FeCl2 </center>**
  
  **Cu2+离子具有氧化性，能与铜发生氧化反应**
  
  **<center>CuCl2 + Cu → 2Cucl </center>**
  
  **所以，FeCl3蚀刻液对铜的蚀刻是靠三价铁离子和二价铜离子共同完成的。**
  
  >由口诀：“**升失氧化还原剂；降得还原氧化剂**”可知，
  
  >在本反应中，三价铁离子和二价铜离子化合价都降低了，即被还原，也就是说该蚀刻液为***氧化剂*** , 具有***氧化性***。

  + 问题2：什么是COB工艺？完成COB工艺需要哪些设备？

  **答：COB（Chip On Board）,即板上芯片，也称为芯片直接贴装技术。是指将裸芯片直接粘贴在印刷电路板上，然后进行引线键合，再用有机胶将芯片和引线包封保护的工艺。**
  
  > 引线键合（Wire Bonding）是一种使用细金属线，利用热、压力、超声波能量为使金属引线与基板焊盘紧密焊合，实现芯片与基板间的电气互连和芯片间的信息互通的手段。
  
  **COB工艺流程及基本要求：清洁PCB → 滴粘接胶 → 芯片粘贴 → 测试 → 封黑胶加热固化 → 测试 → 入库**
  
  **所以，完成COB主要设备有：**
  
        1. 自动线焊机，是用来自动互联键合的。
        
        2. 点胶机，是专门对流体进行控制，并将流体点滴、涂覆于产品表面或产品内部的自动化设备。
        
        3. 烘箱（带鼓风）：带鼓风的烘箱主要作用是将结胶固化。
        
        4. 拉力测试仪：拉力测试仪的主要作用是测试邦线的拉力大小。

  + 问题3：什么是SIP工艺？
  
  **答：SiP模组是一个功能齐全的子系统，它将一个或多个IC（集成电路）芯片及被动元件整合在一个封装中。**
  
  **主要特点如下：**
  
     1. 尺寸小
     
     2. 速度快
     
     3. 成本低
     
     4. 高生产效率
